# Python Socket

This example is for those who start based on Python sockets.

The project includes a basic one-to-one connection, one-to-many connection, and many-to-many connection.

### Project Structure
```
Python-Socket/
├── .idea
├── basic
|    ├── client.py
|    └── server.py
├── many
|    ├── many-client.py
|    └── many-server.py
├── multi
|    ├── multi-client.py
|    └──multi-server.py
└── README.md
```
